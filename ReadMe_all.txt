1. Для работы программа нужно заполнить файл Configuration_Data.xlsx:

	Лист CV: Столбцы - POV, Нижнее ограничение, Верхнее ограничение, Верхнее инженерной ограничение, Нижнее инженерное ограничение - могут
	быть заполнены как числом, так и именем OPC тега соответствующей привязки. 
	Для работы в режиме симуляции необходимо заполнить данные поля значениями, в данном случае поле POV будет начальным значением CV. 
	Для реализации интеграции есть лист с шаблоном.

	Лист MV: Столбцы - PV, SV, RSV, OP, Текущий шаг, Нижнее ограничение, Верхнее ограничение, Верхнее инженерной ограничение, 
	Нижнее инженерное ограничени - могут быть заполнены как числом, так и именем OPC тега соответствующей привязки. 
	Остальные поля заполняются значениями. Для работы в режиме симуляции необходимо заполнить данные поля значениями, 
	в данном случае поле PV будет начальным значением MV. Также для работы в режиме симуляции необходимо заполнить поля
	PV, SV, RSV одинаковыми значениями.

	Для реализации интеграции есть лист с шаблоном.

	Лист DV: Столбец значение - может быть заполнен как числом, так и именем OPC тега соответствующей привязки. Для работы в режиме
	симуляции необходимо заполнить данное поле значением, оно и будет начальным для DV.  

	Лист Matrix: Заполяется по шаблону, который приведен в исходной версии файла Configuration_Data.xlsx. Скрипт автоматические определяет интегральную модель,
	если введено только два коэффициента.
	
	Лист General: Заполняется по примеру

	Срипт автоматические определяет только модели первого порядка и интегральное звено. Флаг интегральности у CV нужно выставлять отдельно.

	
	
	Важно! Имена и количество элементов в столбце Name должны совпадать с именами и количеством переменных на листе Matrix

2. После заполнения файла Configuration_Data.xlsx. Необходимо запустить Make_config.EXE.

3. После выполнения Make_config.EXE в текущей папке сформируется конфигурационный файл ItsYourMPCConfig.xml. Данный конфигурационный файл нужно импортировать в T-Engineering Studio.

4. Для импорта в T-Core необходимо выбрать данный файл в качестве конфигурации в T-Control. Предварительно можно проверить данный файл, импортировав его в T-Engineering Studio
